===============================================================================
一、系统简介
===============================================================================

本系统是一个基于YOLOv8的智能人群检测与人流统计系统，支持普通摄像头和
无人机航拍两种模式，具备实时检测、热力图分析、区域计数、数据导出等
完整功能。

系统架构：
  输入层：摄像头 | 图片 | 视频 | RTSP流
     ↓
  检测层：普通模式(YOLOv8n) | 航拍模式(YOLOv8s)
     ↓
  处理层：目标检测 → 追踪 → 计数 → 热力图 → 区域统计 → 告警
     ↓
  输出层：画面显示 | 截图保存 | CSV报表 | 日志文件


===============================================================================
二、功能特点
===============================================================================

核心功能：
  1. 四种输入源：摄像头、图片、视频、RTSP网络流
  2. 两种检测模式：普通模式（地面）、航拍模式（无人机）
  3. 实时人数统计：显示当前人数、峰值人数
  4. 密度热力图：红黄蓝颜色显示人群密度分布
  5. 累计热力图：长期人群热点累积分析
  6. 区域计数：自定义区域，统计区域内人数
  7. 高密度告警：人数超过阈值自动告警
  8. 截图保存：一键保存当前检测画面
  9. 报表导出：导出CSV格式统计报表
  10. 参数调节：实时调节置信度阈值

技术指标：
  指标              普通模式        航拍模式
  检测速度(CPU)     15-25 FPS       5-10 FPS
  检测速度(GPU)     60-100 FPS      20-40 FPS
  输入分辨率        640×640         1280×1280
  最小可检测目标    30×30像素       15×15像素
  适用高度          地面~10米       10~100米
  模型大小          ~6MB            ~22MB

四、快速安装
===============================================================================

步骤1：安装Python
  Windows：从 https://www.python.org/downloads/ 下载安装
  macOS：brew install python@3.10
  Linux：sudo apt install python3.10 python3-pip

步骤2：下载代码
  将 main.py 保存到项目文件夹

步骤3：安装依赖
  # 基础依赖
  pip install opencv-python==4.8.1.78
  pip install pillow==10.1.0
  pip install numpy==1.24.3
  pip install ultralytics==8.0.200

  # 国内用户使用清华镜像
  pip install -i https://pypi.tuna.tsinghua.edu.cn/simple opencv-python pillow numpy ultralytics

步骤4：验证安装
  python -c "import cv2; print(cv2.__version__)"
  python -c "from ultralytics import YOLO; print('YOLO OK')"

六、操作说明
===============================================================================

6.1 按钮功能说明
  按钮          功能              说明
  ──────────────────────────────────────────────────────────────
  摄像头        启动实时检测      调用电脑摄像头开始检测
  图片          检测单张图片      选择图片文件进行检测
  视频          检测视频文件      选择视频文件进行检测
  普通模式      地面场景检测      速度快，适合手持/固定摄像头
  航拍模式      无人机场景检测    高精度，适合高空俯拍
  置信度滑块    调节检测灵敏度    左调低（漏检少）、右调高（误检少）
  定义区域      画检测区域        只统计区域内的行人
  清除区域      清除画好的区域    恢复全图统计
  停止          停止检测          释放摄像头资源
  截图          保存当前画面      保存为JPG文件
  导出报表      导出统计数据      导出CSV格式报表
  重置统计      重置所有数据      清零人数、峰值等

6.2 场景操作流程

  场景1：摄像头实时检测
    1. 运行程序 → 打开图形界面
    2. 选择检测模式（普通/航拍）
    3. 点击"摄像头"按钮
    4. 等待画面出现，开始检测
    5. 观察左侧画面和右侧统计数据
    6. 按需要调节置信度滑块
    7. 点击"停止"结束检测

  场景2：检测单张图片
    1. 运行程序 → 打开图形界面
    2. 点击"图片"按钮
    3. 在弹出的文件对话框中选择图片
    4. 程序自动检测并显示结果
    5. 可按"截图"保存结果

  场景3：检测视频文件
    1. 运行程序 → 打开图形界面
    2. 点击"视频"按钮
    3. 选择视频文件（mp4/avi/mov/mkv）
    4. 程序自动播放并检测
    5. 点击"停止"结束检测

  场景4：定义检测区域
    1. 启动检测（摄像头/图片/视频）
    2. 点击"定义区域"按钮
    3. 在弹出的OpenCV窗口中：
       - 鼠标左键点击 → 添加顶点（至少3个点）
       - 按Enter键 → 完成定义
       - 按ESC键 → 取消定义
    4. 区域定义后，右侧会显示区域内人数
    5. 点击"清除区域"可清除

八、配置说明
===============================================================================

配置文件位置：项目文件夹/config.json（首次运行后自动生成）

配置项说明：
  {
    "model": {
      "small": "yolov8n.pt",      普通模式模型
      "medium": "yolov8s.pt"      航拍模式模型
    },
    "detection": {
      "conf_threshold": 0.5,      普通模式置信度
      "iou_threshold": 0.5,       普通模式IOU阈值
      "imgsz": 640                普通模式输入尺寸
    },
    "drone_mode": {
      "conf_threshold": 0.35,     航拍模式置信度
      "iou_threshold": 0.4,       航拍模式IOU阈值
      "imgsz": 1280               航拍模式输入尺寸
    },
    "performance": {
      "frame_skip": 1,            跳帧数（1=每帧处理）
      "smooth_window": 5          平滑窗口大小
    },
    "alert": {
      "threshold": 50             告警阈值（人数）
    },
    "output": {
      "save_screenshots": true,   是否自动保存告警截图
      "auto_export": true         退出时自动导出报表
    }
  }

修改配置后需要重启程序生效

九、常见问题
===============================================================================

Q1：程序启动时报错 "No module named 'cv2'"
原因：OpenCV未安装
解决：pip install opencv-python

Q2：点击摄像头后黑屏/无画面
原因：摄像头被占用或驱动问题
解决：
  1. 检查摄像头是否被其他软件占用（关闭其他摄像软件）
  2. 检查摄像头连接是否正常
  3. 尝试重启电脑

Q3：检测速度很慢（FPS低于5）
原因：CPU性能不足或模型太大
解决：
  1. 使用普通模式（不要用航拍模式）
  2. 关闭热力图功能
  3. 修改config.json中的imgsz为更小值

Q4：无法下载模型文件
原因：网络问题或防火墙限制
解决：
  1. 检查网络连接
  2. 手动下载模型放到项目文件夹
  3. 使用镜像源：pip install -i https://pypi.tuna.tsinghua.edu.cn/simple ultralytics

Q5：检测框抖动严重
原因：单帧检测不稳定
解决：
  1. 增加置信度阈值（向右拖动滑块）
  2. 确保光线充足
  3. 保持摄像头稳定

Q6：热力图不显示
原因：热力图功能未开启或人数为0
解决：
  1. 勾选"密度热力图"复选框
  2. 确保画面中有人
  3. 调低置信度阈值

Q7：区域内人数显示为0
原因：区域定义不准确或无人进入
解决：
  1. 重新定义区域，确保覆盖行人位置
  2. 检查区域顶点是否闭合
  3. 确保行人底部中心点在区域内
